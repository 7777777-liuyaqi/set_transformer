# set_transformer
Official Pytorch implementation of the paper 
[Set Transformer: A Framework for Attention-based Permutation-Invariant Neural Networks
](http://proceedings.mlr.press/v97/lee19d.html).

# requirements
- Python 3
- Pytorch >= 1.0
- tqdm
